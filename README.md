# SpringK Portfolio

Welcome to the SpringK Portfolio, an art portfolio website showcasing a collection of acrylic on canvas artworks. This project is built with Next.js and Material UI to provide a responsive and immersive viewing experience.

## Features

- **Splash Screen**: An engaging intro animation that welcomes visitors.
- **Art Gallery**: A responsive grid layout displaying the artwork collection.
- **Image Preview**: Interactive modal (lightbox) for viewing high-resolution details of each piece.
- **Responsive Design**: Optimized for seamless viewing on mobile, tablet, and desktop devices.
- **Navigation**: Intuitive navigation with a responsive hamburger menu for mobile devices.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org)
- **UI Library**: [Material UI (MUI)](https://mui.com)
- **Styling**: Emotion
- **Deployment**: [Cloudflare Pages](https://pages.cloudflare.com)

## Project Structure

- `src/app`: Contains the application routes and page logic (Home, About, Contact).
- `src/components`: Reusable UI components (e.g., ArtCard, SplashScreen, Navbar).
- `src/data`: Static data files, including the `artworks` collection.
- `public/assets`: Static image assets for the portfolio.

## Getting Started
